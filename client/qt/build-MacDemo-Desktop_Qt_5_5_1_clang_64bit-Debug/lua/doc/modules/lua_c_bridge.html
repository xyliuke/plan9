callback (param, result, data, data_style)
这个函数供lua调用,用来将值回调给c++

Parameters:

  • param 由c++传递来的参数
  • result 回调的成功或失败,值必须为boolean值或者nil值,如果为nil则为false
  • data 回调给C++的数据圣贤,必须为table值或nil值
  • data_style 检测data是否符合data_style定义的格式,可以为nil
